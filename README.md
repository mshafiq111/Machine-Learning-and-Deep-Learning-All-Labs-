# Machine Learning & Deep Learning Lab


Welcome to the **Machine Learning and Deep Learning Lab Repository**. This repository contains laboratory assignments, practical implementations, and experiments completed as part of the Machine Learning and Deep Learning course.

The primary objective of this repository is to provide hands-on experience with modern Machine Learning (ML) and Deep Learning (DL) techniques using Python and industry-standard data science libraries.

---

## Course Overview

This repository covers a comprehensive range of topics, including:

* Data Analysis and Visualization
* Data Cleaning and Preprocessing
* Feature Engineering
* Data Transformation Techniques
* Regression Models
* Gradient Descent Optimization
* Overfitting and Underfitting Analysis
* Image Dataset Preparation
* Convolutional Neural Networks (CNNs)
* Data Augmentation Techniques
* Multi-Layer Perceptrons (MLP)
* One-Dimensional CNNs (1D CNNs)
* Long Short-Term Memory Networks (LSTM)

---
